    public handle_data_saving(trial: *) source

    Handles saving the relevant data on a given trial.

    For reference, these are the helper variables created to assist in trial logic (i.e not present in excel) trial_variables = {type: 'jnd', run_type: '', left_correlation: '', right_correlation: '', };

    These are variables created WITHIN the trial logic that were not present in excel (but need to be outputted to results). export_variables = {sub_condition: '', // Chronological ordering of sub_condition [1, 2, 3 ... ] balanced_sub_condition: '', // Index of sub_condition according to balancing order jnd: '', base_correlation: '', adjusted_value: '', correct: '', };
