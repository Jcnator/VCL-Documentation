    Correlation_Dependencies

    You can directly use an instance of this class. correlation_dependencies

    Dependencies for Correlation-type experiments such as JND, Sequencing Symbols, and Hybrid. These dependencies handle common behaviors in these experiments related to working with correlation and jnd-tasks. For future expansion/implementation of new features, consider if that feature is unique to the experiment it is being developed for, or if multiple experiments (current or not) could benefit from it.

    Correlation experiments that use these dependencies need the following attributes (depending on which functions it will use):

    EXPERIMENT PROPERTIES:

    • base correlation
    • initial difference
    • max_step_size
    • direction

    EXPERIMENT OBJECT ATTRIBUTES

    • converged
    • adjusted_quantity_matrix
    • adjusted_quantity_practice
